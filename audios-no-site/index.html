Reproduzindo Áudios

Exemplficando como reproduzir áudios em um site.

Evitar o preload AUTO, pois carrega todo o áudio.

O preload METADATA carrega só algumas informações.

O preload NONE não carrega nenhuma informação.

Há outros argumentos para o PRELOAD.

Podemos colocar áudios tipo MP3, OGG e WAV

Os áudios são sempre carregados na ordem de cima para baixo.

AUTOPLAY se quizermos que o áudio inicie com o site.

LOOP se quizermos que continue sempre reproduzindo.

Evitar áudios no formato WAV, pois são muito pesados.
