Exibição/Formatação de Fontes e Citações

Fontes

Modificando uma fonte que não é mono-espaçada, ou seja, cada caracter não tem o mesmo espaço. Para ajustar esses espaços, quando necessário, formatamos com a tag CODE.

Sem a tag CODE: document.getElementById('teste')

Com a tag CODE: document.getElementById('teste')

Se, por exemplo, num código, só usarmos a tag CODE ela desorganiza a IDENTAÇÂO.

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é par') else: print(f'O número {num} é impar') print('Fim do programa')

Para preservar a identação devemos envelopar tudo dentro da tag PRE. 

        
            num = int(input('Digite um número'))
            if num % 2 == 0:
                print(f'O número {num} é par')
            else:
                print(f'O número {num} é impar')
            print('Fim do programa')

Se o código ficou muito para frente, podemos tabulá-lo para trás (SHIFT+TAB)

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é par')
    else:
        print(f'O número {num} é impar')
    print('Fim do programa')

Citações Simples

Dizemos que tudo "o Freud explica". Aqui temos somente a citação como FORMA, não há SEMÂNTICA.

O correto é usarmos as citações com SEMÂNTICA. Para isso a tag é Q (quote)

Dizemos que tudo o Freud explica.

Quando for uma citação sem importância, usa-se aspas apenas.
