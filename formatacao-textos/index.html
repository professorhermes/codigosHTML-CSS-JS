Algumas Formatações com SEMÂNTICA

Negrito - Destaque

Nesta frase aparece um termo em negrito usando a tag B, que é somente FORMA e não possui SEMÂNTICA.

Já nesta frase vemos um termo em negrito usando a tag STRONG, que além de ter FORMA tem valor SEMÂNTICO.

Itálico - Ênfase

Nesta frase temos um termo em itálico usando a tag I, que é somente FORMA e não possui valor SEMÂNTICO.

Nesta, usamos o termo em itálico usando a tag EM, que além de ter FORMA tem SEMÂNTICA. EM (ênfase).

Texto Marcado

Podemos marcar um texto com um destaque colorido. Usa-se, para isto, a tag MARK.

Texto Pequeno

As vezes necessitamos, como uma observação, por exemplo, de uma palavra ou parágrafo com letras pequenas. Usamos, então, a tag SMALL, que possui SEMÂNTICA.

Texto Deletado/Excuído

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado. Usamos a tag DEL, que possui valor SEMÂNTICO.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usamos a tag INS, que possui valor SEMÂNTICO.

Texto Sobrescrito

Para inserir coisas do tipo: x2 + 3. O 2 deve ficar acima do x. Usamos a tag SUP.

Texto Subscrito

Para inserir coisas do tipo: H2O. O 2 deve ficar abaixo do h. Usamos a tag SUB.
