Usando o captura de textos NINJA

Este código é para somente explicar como capturamos uma fonte de uma página que vimos na internet.

Primeiro instalamos do CHROME WEB STORE uma extenção chamada FONTS NINJA.

Essa extenção fica instalada na linha superior do navegador Google Chrome, junto com as outras extenções WEB DEVELOPER e o COLOR ZILLA, estes dois são importantes para o trabalho de desenvolvimento.
