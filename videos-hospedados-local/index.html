Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu próprio servidor.

O argumento POSTER é para que quando se atualiza o vídeo ele coloca uma imagem que selecionamos previamente. É uma opção elegante.

A ordem é sempre de cima para baixo. Se não carrega o primeiro, tenta o de baixo etc.

Se nenhum for compatível com o navegador ele dará a mensagem do parágrafo abaixo.

Podemos colocar vários formatos do MESMO vídeo para que se possível possa ser executado em qualquer navegador. Pode-se colocar também vídeos diferentes se não houver obrigatoriedade de um determinado vídeo.

AUTOPLAY e LOOP são opcionais.

ATENÇÃO: VÍDEOS HOSPEDADOS LOCALMENTE NO SERVIDOR GERA UM CUSTO ELEVADO AO CLIENTE POR CAUSA DO TRÁFEGO DE BANDA, PRINCIPALMENTE SE FOR MUITO ACESSADO. SE O CLIENTE QUIZER PODE SER UTILIZADO VÍDEOS EXTERNOS.
